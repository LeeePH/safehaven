A beautiful React application with a secret calculator that reveals animated flowers when the correct code is entered.

## Features

- **Welcome Page**: A beautiful greeting page with a functional calculator
- **Secret Code**: Enter "143" and press "=" to unlock the flower animation
- **Animated Flowers**: Beautiful CSS-animated flowers that bloom and move
- **Modern UI**: Built with React, Tailwind CSS, and Vite

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## How to Use

1. You'll see a welcome page with a greeting and a calculator
2. Type the secret code: **143**
3. Press the **=** button
4. You'll be redirected to the beautiful flower animation page!

## Tech Stack

- **React 18** - UI library
- **React Router** - Navigation
- **Tailwind CSS** - Styling
- **Vite** - Build tool and dev server
